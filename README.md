# ⚾ Angry Baseball — 第一人稱 3D 憤怒棒球

用 **three.js** 渲染、**cannon-es** 3D 物理打造的第一人稱打擊遊戲（[angry-pig](../angry-pig) 姊妹作，沿用同一套技術與美術）。投手把球投過來，抓準時機揮棒，看球飛越外野牆——依距離與全壘打計分，含全球排行榜、球棒商店、可破壞的外野目標與 JACKPOT 飛天金豬。

## 遊戲模式

| 模式 | 說明 | 排行榜 |
|------|------|--------|
| ⚾ **全壘打大賽** | 10 球定勝負，第 5、9 球必出 ✨黃金球（×2） | ✅ |
| 🔥 **生存連打** | 揮空／看好球 3 次三振出局；球速球種越來越刁鑽，每 10 球一顆 🔥火球（×3） | ✅ |
| 📖 **投手挑戰** | 12 關逐關解鎖，依分數給 1~3 星（狼投手、火球手、Boss 傳說對決） | 各關榜 |
| 🚀 **最遠紀錄** | 所有模式的單球最遠距離，獨立排行榜 | ✅ |

## 玩法

- **移動滑鼠**：瞄準（準星壓球心下緣＝高飛球、正中＝平飛、上緣＝滾地）
- **左鍵點擊**：揮棒——白色縮圈貼合好球帶的瞬間＝PERFECT（金屬鏗聲 + 頓幀）
- **時機決定方向**：提早揮＝拉打左外野、晚揮＝推打右外野（左右邊線牆只有 92m！）
- **M**：音樂　**Esc**：暫停　**手機**：左下搖桿瞄準、右下「揮棒」鈕
- 擊球品質 ≥ 強勁 → **Ball-Cam 慢動作追球**，看球飛出牆外
- 連續安打累計 **Combo 倍率**（最高 ×2），揮空歸零

## 球棒商店（金幣跨場累積）

木棒（免費）→ 竹棒（超大甜蜜點）→ 鋁棒 → 鹹魚（搞笑亂飛）→ 大鎚（打中必轟天、甜蜜點極小）→ 火焰棒（無空氣阻力 +25% 距離）→ 磁力棒（顯示進壘預測點）→ 雷神棒（PERFECT 落雷）→ 傳說金棒（全屬性頂級）。
金幣來源：安打 +20、全壘打 +100、PERFECT +10、場上目標賞金 ÷10。

## 開發

```bash
npm install
npm run dev      # http://localhost:5181
npm run build    # 打包到 dist/
node test/smoke.mjs   # 無頭瀏覽器冒煙測試（需本機 Chrome；先啟動 dev server）
```

## 技術

| 項目 | 技術 |
|------|------|
| 渲染 | three.js r160（第一人稱球棒視角、Ball-Cam、程序貼圖草皮/棒球縫線） |
| 物理 | cannon-es 0.20（投球用解析初速 + 每步變化球加速度模擬 Magnus 力；擊球判定＝時機+準星混合，轉衝量交給物理） |
| 音效 / 音樂 | Web Audio 即時合成（球棒鏗、手套啪、群眾歡呼隨距離漸強）+ 背景音樂 mp3 |
| 建置 | Vite 5 |
| 後端 | Cloudflare Pages Functions + D1（排行榜 / 在線 / 留言板 / 統計，IP 限流 + 分數合理性檢查防灌分） |

### 檔案

- `index.html` — HUD + 登入頁 + 選單 + 商店/排行榜/留言板彈窗 + 樣式
- `game.js` — 主程式（投打狀態機、模式流程、Ball-Cam、商店、排行榜客戶端）
- `stadium.js` — 渲染核心 + 球場幾何（外野牆弧線、看台、觀眾動物、4 種 biome 換景）
- `batting.js` — 揮棒判定純函式 + 球棒數據表
- `pitch`（在 game.js）— 7 種球種：直球/慢速/曲球/滑球/變速/蝴蝶/火球
- `fx.js` — 跳字 / 光點 / 爆炸 / 煙火 / 落雷
- `sfx.js` — Web Audio 音效與音樂
- `functions/api/` — Cloudflare Pages Functions（與 angry-pig 同架構）
- `schema.sql` — D1 結構

## 部署（Cloudflare Pages）

```bash
wrangler login
wrangler d1 create angry-baseball-db      # 把 database_id 填入 wrangler.jsonc
npm run db:init                            # 套用 schema 到遠端 D1
npm run deploy                             # build + 部署
```

開發時無 `/api` 後端會自動回退 localStorage，單機仍可完整遊玩。
